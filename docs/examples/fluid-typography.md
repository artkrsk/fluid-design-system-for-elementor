# Fluid Typography Examples

Real-world examples of using fluid typography in your Elementor designs.

## Heading Scale

Create a complete typography scale that adapts smoothly:

| Element | Preset | Min Size | Max Size |
|---------|--------|----------|----------|
| H1 | Display | 36px | 96px |
| H2 | Heading Large | 28px | 64px |
| H3 | Heading Medium | 24px | 48px |
| H4 | Heading Small | 20px | 36px |
| Body | Body Text | 16px | 18px |

## Implementation

Apply these presets in Elementor's Typography controls by selecting the **fluid** unit and choosing your preset.

See [Creating Presets](/guides/creating-presets) for setup instructions.
