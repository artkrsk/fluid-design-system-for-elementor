# Fluid Spacing Examples

Real-world examples of using fluid spacing for padding, margins, and gaps.

## Spacing Scale

| Preset | Min Size | Max Size | Use Case |
|--------|----------|----------|----------|
| Space XL | 80px | 200px | Major sections |
| Space L | 40px | 120px | Section padding |
| Space M | 24px | 64px | Element spacing |
| Space S | 16px | 32px | Small gaps |

## Common Patterns

- **Section padding**: Use Space L or XL
- **Element margins**: Use Space M
- **Grid gaps**: Use Space S or M
- **Card padding**: Use Space S

See [Creating Presets](/guides/creating-presets) for setup instructions.
