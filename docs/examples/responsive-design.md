# Responsive Design Patterns

Best practices for creating responsive layouts with fluid design.

## Mobile-First Approach

Start with mobile sizes (minimum values) and scale up to desktop (maximum values):

1. Design for 360px (mobile)
2. Set maximum for 1920px (desktop)
3. Everything in between scales automatically

## Common Patterns

### Hero Section
- Heading: 36px ~ 96px
- Padding: 80px ~ 200px
- Gap: 24px ~ 64px

### Content Section
- Heading: 28px ~ 64px
- Padding: 40px ~ 120px
- Gap: 16px ~ 48px

### Card Grid
- Padding: 24px ~ 48px
- Gap: 16px ~ 32px
- Border radius: 8px (static)

See [Fluid Typography](/examples/fluid-typography) and [Fluid Spacing](/examples/fluid-spacing) for more examples.
