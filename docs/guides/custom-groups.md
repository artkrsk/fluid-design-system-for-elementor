# Custom Groups

Organize your fluid presets into custom groups for better management and organization.

## Creating a Group

1. Navigate to **Elementor > Fluid Design System** in WordPress admin
2. Click **Add New Group**
3. Enter a group name (e.g., "Border Radius", "Thumbnail Sizes")
4. Add presets to the group using drag-and-drop

## Managing Groups

- **Drag and drop** presets between groups
- **Search** for specific presets
- **Delete** groups when no longer needed
- **Rename** groups for better organization

See [Getting Started](/getting-started) for basic usage.
