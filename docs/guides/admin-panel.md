# Admin Panel

Access the comprehensive management interface for organizing your fluid design presets.

## Accessing the Panel

Navigate to **Elementor > Fluid Design System** in your WordPress admin dashboard.

## Features

- **Cross-group management** - Drag presets between groups
- **Search functionality** - Find presets quickly
- **Preset organization** - Create custom groups for different use cases
- **Real-time synchronization** - Changes sync immediately

See [Custom Groups](/guides/custom-groups) for detailed usage.
